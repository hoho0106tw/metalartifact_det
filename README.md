本研究針對 CT 灰階影像中的高亮度金屬區域進行自動化偵測，主要結合 Otsu 灰階閾值法與連通元件分析（connected component analysis），以找出可能的金屬夾位置。整體流程如下：

首先，將輸入影像以灰階格式讀入，並針對高亮度區域進行分析。考量 CT 影像中的金屬物質通常具有較高灰階值，本研究先設定 min_val = 220 作為亮區下限，僅擷取灰階值大於此門檻的像素進行統計。接著，利用這些亮區像素建立灰階直方圖，並在高亮區範圍內套用 Otsu thresholding method，尋找可使前景與背景之類別間變異數（between-class variance）最大的最佳灰階閾值。此作法可避免整張影像受到大量低灰階背景影響，使閾值更聚焦於金屬亮點的分布特性。

在取得最佳灰階閾值後，將原始 CT 影像進行二值化處理：凡灰階值大於或等於該閾值者標記為白色（255），其餘標記為黑色（0），以產生金屬候選區域的二值影像。此步驟可將金屬高亮區域自背景與一般組織中分離出來，作為後續偵測的基礎。

接下來，針對二值影像進行 8-鄰域連通元件分析，擷取所有獨立白色區塊的統計特徵，包括位置座標、寬度、高度、面積及質心位置。為了篩選出符合金屬夾外觀特徵的候選物件，本研究進一步設定多項條件限制，包括：

物件面積需介於設定範圍內（如 20 至 2000 像素），以排除過小雜訊與過大非目標區域；
物件質心需位於影像上半部，因金屬夾通常分布於影像較上方位置；
物件須接近影像左側或右側區域，以符合金屬夾常位於兩側邊緣的影像特性；
物件寬高需小於一定範圍，避免過大區塊被誤判；
長寬比需落於合理範圍內，以保留符合金屬夾形狀特徵的區域。

符合上述條件的連通元件即視為偵測到的金屬夾，並於原始影像上繪製外接框與標記名稱，以利視覺化檢視與後續分析。此外，系統亦輸出每個偵測結果的邊界框座標、面積與質心位置，作為定量評估依據。

整體而言，本方法透過亮區限定的 Otsu 閾值計算，提升高亮金屬區域分割的準確性，再配合位置、大小與形狀條件篩選，可有效從 CT 影像中擷取疑似金屬夾區域，降低人工判讀的主觀性，並提升偵測流程的自動化程度與一致性。
